
- 'node' 是 TypeScript 社区中最常用的，建议大多数项目使用。此解析策略试图在运行时模拟 Node.js 模块解析机制。
- 'node16' 或 'nodenext', 从 TypeScript 4.7开始支持 Node.js 的 ECMAScript 模块
- 'classic'，在1.6版本发布之前的 TypeScript 中使用。在现代代码中，您可能不需要使用它
